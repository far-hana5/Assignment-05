What is the Document object Model (DOM)

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page (HTML or XML) as a tree of objects, allowing programs and scripts (like JavaScript) to dynamically access, modify, and manipulate the content, structure, and styles of the document.

How do you select element from DOM?

Here’s a brief explanation of how to select elements from the DOM: By ID: Select an element using its unique id. By Class Name: Select all elements that have a specific class. By Tag Name: Select all elements with a specific HTML tag (like div or p). By Query Selector: Select the first element that matches a CSS selector. By Query Selector All: Select all elements that match a CSS selector. By Name Attribute: Select elements using their name attribute.

What is event delegation in the context of dom, and why it is useful?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element to manage events from its child elements. Instead of adding multiple listeners to each child, the event bubbles up to the parent, which handles it.
Why it's
Improves performance by reducing the number of event listeners.
Handles dynamically added elements without needing to attach new event listeners to them.

How do you manipulate an element an elements of atributes and style using the DOM?

You can manipulate an element’s attributes and style using the DOM in the following
Manipulate Attributes:

Use setAttribute() to set or change an attribute.
Use getAttribute() to retrieve an attribute's value.
Use removeAttribute() to remove an attribute.
Manipulate Style:
Access the element's style property and modify CSS properties directly.
Example: element.style.backgroundColor = 'blue'; changes the background color
